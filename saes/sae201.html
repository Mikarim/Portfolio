 SAE201

construire un réseau informatique pour une petite structure

L'objectif de ce projet était de créer un réseau informatique utilisant OpenStack, comprenant deux serveurs Linux et deux machines Windows. Chaque machine était capable de communiquer avec chacune d'entre elles et possédait ses propres fonctionnalités.

serva : Linux FTP, DNS, Prometheus
servb : Linux Apache, Nextcloud
w10a : Windows 10
w10b : Windows 10
