 SAE201

construire un réseau informatique pour une petite structure

Ce projet consistait à l'élaboration d'un réseau informatique avec OpenStack, composé de deux serveurs linux et de deux machines windows. Toutes les machines pouvaient communiquer avec chacunes.

serva : Linux FTP, DNS, Prometheus
servb : Linux Apache, Nextcloud
w10a : Windows 10
w10b : Windows 10
